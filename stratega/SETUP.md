# Инструкция по установке и запуску

## Быстрый старт

1. **Установите зависимости:**
```bash
npm install
```

2. **Настройте переменные окружения:**
Создайте файл `.env` на основе `.env.example`:
```bash
DATABASE_URL="file:./dev.db"
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=your-secret-key-here
```

3. **Инициализируйте базу данных:**
```bash
npm run db:generate
npm run db:push
```

4. **Опционально: Заполните тестовыми данными:**
```bash
npm run db:seed
```

5. **Запустите dev сервер:**
```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

## Структура проекта

- `/app` - страницы Next.js (App Router)
  - `/catalog` - каталог товаров
  - `/services` - услуги
  - `/admin` - админ-панель
  - `/api` - API endpoints
- `/components` - React компоненты
- `/lib` - утилиты и конфигурация
- `/prisma` - схема базы данных

## Основные страницы

- `/` - Главная
- `/catalog` - Каталог товаров
- `/services` - Услуги
- `/about` - О компании
- `/contacts` - Контакты
- `/admin` - Админ-панель

## Админ-панель

**Внимание:** В текущей версии аутентификация не реализована. 
Добавьте её перед деплоем в продакшен!

Доступные разделы:
- `/admin` - Главная панель со статистикой
- `/admin/products` - Управление товарами
- `/admin/services` - Управление услугами
- `/admin/categories` - Управление категориями
- `/admin/requests` - Обработка заявок

## Следующие шаги

1. **Добавьте аутентификацию:**
   - Настройте NextAuth.js
   - Создайте первого администратора
   - Защитите админ-роуты

2. **Настройте email уведомления:**
   - Заполните SMTP настройки в `.env`
   - Реализуйте отправку уведомлений о новых заявках

3. **Добавьте загрузку файлов:**
   - Настройте хранение изображений и документов
   - Можно использовать локальное хранилище или S3

4. **Настройте SEO:**
   - Добавьте Google Analytics ID
   - Настройте мета-теги для каждой страницы
   - Проверьте sitemap.xml

5. **Для продакшена:**
   - Перейдите на PostgreSQL
   - Настройте резервное копирование БД
   - Добавьте rate limiting
   - Настройте reCAPTCHA
   - Включите HTTPS

## Полезные команды

```bash
# Генерация Prisma Client
npm run db:generate

# Применение изменений схемы БД
npm run db:push

# Создание миграции
npm run db:migrate

# Открыть Prisma Studio (GUI для БД)
npm run db:studio

# Заполнить БД тестовыми данными
npm run db:seed

# Сборка для продакшена
npm run build

# Запуск продакшен версии
npm start
```

## Поддержка

При возникновении проблем проверьте:
1. Установлены ли все зависимости (`npm install`)
2. Создан ли файл `.env` с правильными настройками
3. Инициализирована ли БД (`npm run db:push`)
4. Запущен ли dev сервер (`npm run dev`)
